# 可爱小鸟飞游戏

一个基于HTML5 Canvas的休闲小游戏，支持PC和移动设备。

## 游戏介绍

这是一个类似Flappy Bird的休闲小游戏，玩家通过点击或触摸屏幕控制小鸟飞行，穿越管道间隙，收集水果获取高分。

## 游戏功能

- ✨ 精美的游戏画面
- 📱 支持移动设备触摸操作
- 🎵 丰富的音效和背景音乐
- 🏆 得分系统
- 📱 自动适配竖屏和横屏
- 🚀 图片自动压缩，加载更快

## 如何玩

1. **开始游戏**：点击「开始游戏」按钮
2. **控制小鸟**：点击或触摸屏幕使小鸟向上飞行
3. **穿越管道**：控制小鸟穿过管道之间的间隙
4. **收集水果**：收集水果可以获得额外分数
5. **挑战高分**：尽可能飞得更远，获得更高的分数

## 技术特点

- 使用HTML5 Canvas绘制游戏画面
- 响应式设计，适配不同屏幕尺寸
- 图片自动压缩，提升加载速度
- 支持触摸和鼠标操作
- 渐进式资源加载，游戏可以在资源加载完成前开始
